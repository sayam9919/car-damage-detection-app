# Car Damage Detection and Repair Cost Estimation

This project detects car damage from an uploaded image and estimates repair cost using YOLOv8.

## Features

- Detects car damages
- Identifies damage type
- Estimates repair cost
- Web interface using Streamlit

## Tech Stack

Python  
YOLOv8  
Streamlit  
OpenCV  

## Run Locally

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py
